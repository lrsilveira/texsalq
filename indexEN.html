Welcome to TeXSALQ page

TeXSALQ is a set of macros and templates made in plain TeX (not LaTeX), that will be soon officially recommended to masters and doctoral students for the preparation of their theses and dissertations at ESALQ/USP. It is based on the template CTUStyle developed by Petr Olsak, professor at the Czech Technical University in Prague; and uses at its base the OPmac macros, from the same author.

To use this style you should have a basic knowledge of plain TeX and a functional version of pdfTeX. The program pdfTeX is part of TeXlive package available in most Linux distributions. TeXlive is freely available, and can also be installed on Macintosh or MS Windows. However, for MS Windows it is recommend MikTeX program.

Instructions

Download the TeXSALQ and save it in a directory of your choice. Unzip the file. Then you must modify the batch file:

For Windows:

Change the name of the file make.bat~ to make.bat

For Linux:

Make the file make.sh executable: chmod +x make.sh
